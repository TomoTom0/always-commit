# ドキュメント構成

## ディレクトリ構成

```
docs/
├── README.md          # このファイル：ドキュメント構成と更新方針
├── usage/
│   ├── index.md               # ユーザーガイド（コマンドリファレンス、使用例）
│   └── agent-integration.md   # Claude Code との連携ガイド
├── dev/
│   └── index.md       # 開発者ガイド（アーキテクチャ、状態管理、テスト）
└── design/
    └── init.md        # 初期設計ドキュメント（変更不要）
```

## 常時更新が必要なドキュメント

### docs/usage/index.md

**更新タイミング**:
- 新しいコマンドを追加したとき
- 既存コマンドの動作・オプションを変更したとき
- セッション管理の仕様を変更したとき
- インストール手順を変更したとき

### docs/usage/agent-integration.md

**更新タイミング**:
- Claude Code との連携方法（hook設定・settings.json）を変更したとき
- `alcom setup` コマンドの動作・オプションを変更したとき
- hookスクリプト（`scripts/claude-code-hook.sh`）の内容を変更したとき

### docs/dev/index.md

**更新タイミング**:
- アーキテクチャ（コンポーネント構成）を変更したとき
- 状態管理の仕様（ファイルパス、データ構造）を変更したとき
- テスト方法を変更したとき
- 開発環境のセットアップ手順を変更したとき

## 更新不要なドキュメント

### docs/design/init.md

初期設計ドキュメント。過去の意思決定の記録として保持し、変更しない。
