# v1.1.0 (2026-03-23)

初回npmリリース。

## 追加

- `setup` コマンド: Claude Code との連携設定（hookスクリプト配置・settings.json登録）を自動化
- `docs` コマンド: CLIからドキュメントを表示
- worktreeサポート: 状態ファイルのパス優先度を追加（`.git/always-commit.json` を最優先）
- `finish` コマンド: セッションが存在しない場合も通常コミットとして動作

## 修正

- `Bun.spawn` を `child_process.spawn` に置換してNode.js互換性を確保
- root commit（空のツリー）時の `finish` エラーを修正
- テストの並行実行に対応

## その他

- npmパッケージとして公開 (`always-commit`)
- バイナリ: `always-commit`, `alcom`
